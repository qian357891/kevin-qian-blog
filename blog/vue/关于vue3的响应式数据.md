```js
let i = ref(1); 
const demo = () => {
  i.value ++;
  console.log(i);
}
```

```vue
<div>
  <button @click="demo">+</button>
  <div>{{i}}</div>
</div>
```

ref将数据变成响应式数据，进行数据操作时需要用value属性来操作。
