win10的docker desktop内置docker-compose和cli等插件

docker-compose.yml 示例：

注意：`-`与内容之间有一个空格

```
version: '3'

services:
  db:
    image: postgres
    restart: always
    ports:
      - "5432:5432"
    environment:
      POSTGRES_PASSWORD: 123456
```

运行：

```
docker-compose up
```
