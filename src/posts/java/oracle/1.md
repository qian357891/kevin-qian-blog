创建表空间

```sql
CREATE tablespace stx
datafile 'D:\oracle-dbdata\stx.dbf'
SIZE 128m  
autoextend ON
NEXT 16m;
```



创建用户名

```sql
CREATE USER frank_24
IDENTIFIED BY frank_24
DEFAULT tablespace stx;
```



给用户授权

```sql
GRANT dba TO frank_24;
```
